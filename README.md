# AI Vibe-Coded Web Piano

This repository contains a highly optimized master prompt designed to "vibe code" a fully functional, responsive, 5-octave web piano using modern AI coding assistants.

The prompt instructs the AI to build a Vanilla HTML/CSS/JS single-page application utilizing **Tone.js** for audio synthesis, complete with a minimalist glassmorphism UI, mobile-responsive viewport constraints, swipeable octave navigation, and raw `.wav` audio recording capabilities.

## 🚀 How to Use

1. Copy the contents of [`PROMPT.md`](PROMPT.md).
2. Paste the prompt into your preferred AI coding assistant (e.g., Claude Code, Gemini CLI, or a standard chat interface).
3. Allow the AI to generate the HTML, CSS, and JS files.
4. Serve the generated files using a local development server (like VS Code Live Server or `npx serve`) to ensure the browser's Web Audio API and download features function correctly.
5. Click anywhere on the screen to initialize the Audio Context, and start playing!

## 🛠️ Features Generated by the Prompt

* **5-Octave Polyphonic Engine**: Handled via Tone.js.
* **Responsive Viewport**: Automatically restricts the view to 1 octave in portrait mode and 2 octaves in landscape mode to preserve playability on mobile.
* **Swipe Navigation**: Intuitive swipeable ribbon to shift octaves dynamically.
* **Multi-Input Support**: Full support for multi-touch (chords on mobile), mouse clicks, and QWERTY keyboard mapping that dynamically adjusts to the currently visible octave.
* **Classic Instrument Synths**: Automated approximation of classic keyboards (Rhodes, Juno 106, DX7, etc.) via Tone.js synths.
* **Audio Recording**: Built-in buffer capture to download your sessions directly as `.wav` files.
